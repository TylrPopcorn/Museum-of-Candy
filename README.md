# ``Museum-of-Candy``
```A simple practice project utilizing BootStrap HTML. ```

This repository features "Museum-of-Candy," a simple practice project utilizing Bootstrap. The hero page showcases a playful and colorful design inspired by candy themes, featuring repeated text elements and vibrant images. The navigation bar includes links to Candy, Home, About, and Tickets, providing a structured and engaging user experience. This project serves as a creative exercise in implementing Bootstrap HTML for visually appealing web design.
